Add a shortcode for an Avgrund-style modal popup.

== Description ==
Add a shortcode for an [Avgrund-style](https://github.com/voronianski/jquery.avgrund.js) modal popup.

You can add arbitrary markup in the `text` parameter of the shortcode, so be careful.

== Installation ==

1. Upload `avgrund` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Example ==

`[avgrund text='<button>Click Me</button>']This is displayed in a popup.[/avgrund]`
